System Overview

This is the **AI Surveillance Command Center**. It uses **YOLOv8** for real-time object detection (Intruders, Weapons) and the **Flask** server for streaming and alerting.

Key Features
  • Live Feed: Shows real-time detection with bounding boxes.
  • Alerts: Triggers immediate email alerts upon Weapon/Intruder detection (with cooldown).
  • Logs: Logs all events to a CSV file and displays them below.
  • Map: Simulates the location of the detected incident.
Troubleshooting

If the feed is blank, ensure no other application is using your webcam, and check the VS Code terminal for the OpenCV error code (-1072875772).
